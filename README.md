# Fasta-Fastq-Converter

Fastq-Fasta Converter is a desktop application that converts Fastq files into Fasta files.

This project is developed by 1keans: github.com/1keans

# Guide for installation;

First of all, you should download the folder named 'fastq_fasta_converter' from Souceforge link below:

https://sourceforge.net/projects/fastq-fasta-converter/

The program does not need installation. Open the folder 'fastq_fasta_converter' and find the executable file named 'main.exe' by following that directory,
'.venv/Scripts/dist/main.exe';

# Guide for usage;

When you find the 'main.exe' file, double-click on it and the program runs. The program has a user-interface just like this;

<img width="450" height="399" alt="gui" src="https://github.com/user-attachments/assets/e07c45df-d71d-4bcf-a66a-4fd064b538c4" />

You can enter your Fastq to the upper box manually or you can select the Fastq file by clicking the 'Select FASTQ File' button.

<img width="451" height="401" alt="3" src="https://github.com/user-attachments/assets/27f43427-2d76-4ca0-96cf-26ff3dcebb3f" />

After that, click 'Convert to FASTA' button and your Fasta results will appear on the text box below. If you like you can save your Fasta results by clicking 'Save to Word' button.

<img width="407" height="359" alt="4" src="https://github.com/user-attachments/assets/6edcdb4e-63f5-49b9-ba1a-df383765e2bf" />

Do not forget!! The program also do the filtering process. If your Fastq reading quality under the treshold value, the program will not convert your Fastq to Fasta.
